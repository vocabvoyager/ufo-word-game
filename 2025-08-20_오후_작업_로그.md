### 새로운 Unit 생성 과정
1. **단어 데이터 준비**: `data/unitN.json` 파일 생성
2. **생성 스크립트 수정**: `create_unitN.py`로 파일명 변경
3. **자동 생성 실행**: `python create_unitN.py`
4. **완성**: 완전한 게임 폴더와 파일들 자동 생성

### 예시 (Unit 6 생성)
```python
# create_unit6.py
data_path = Path("data") / "unit6.json"
generator.generate_unit(6, unit6_data)
```
